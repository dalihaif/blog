# 🚀 快速开始指南

## 第一步：本地预览（5分钟）

### Windows 用户

1. **双击运行** `启动博客.bat`
2. 等待依赖安装完成
3. 浏览器打开：http://localhost:4000/blog/

### 如果提示没有 Ruby

1. 下载 Ruby Installer：https://rubyinstaller.org/downloads/
2. 选择最新版本（Ruby 3.x + DevKit）
3. 安装时勾选 "Add Ruby to PATH"
4. 重新运行 `启动博客.bat`

---

## 第二步：查看效果

访问以下页面：

- **首页**：http://localhost:4000/blog/
- **关于我**：http://localhost:4000/blog/about.html
- **作品集**：http://localhost:4000/blog/portfolio.html

---

## 第三步：部署到 GitHub（10分钟）

### 1. 初始化 Git 仓库

```bash
cd g:\web\blog-new
git init
git add .
git commit -m "Initial commit: 老墨的个人博客"
git branch -M main
git remote add origin https://github.com/dalihaif/blog.git
git push -u origin main
```

### 2. 启用 GitHub Pages

1. 访问：https://github.com/dalihaif/blog/settings/pages
2. 在 "Build and deployment" 部分：
   - Source: 选择 "GitHub Actions"
3. 点击 "Save"

### 3. 等待自动部署

- GitHub Actions 会自动构建（约 2-5 分钟）
- 查看进度：https://github.com/dalihaif/blog/actions
- 部署成功后访问：https://dalihaif.github.io/blog/

## 遇到问题？

### Q: 本地预览空白或样式错误？
A: 确保使用 `--baseurl "/blog"` 参数启动

### Q: GitHub Pages 不更新？
A: 
1. 检查 GitHub Actions 是否成功运行
2. 清除浏览器缓存（Ctrl+F5）
3. 等待 5-10 分钟

### Q: 如何修改个人信息？
A: 编辑 `_config.yml` 和 `about.md` 文件
