## 一、Quicker 是什么？先想象一个画面

你正在写一篇报告，需要从一张截图里把文字抄下来。

正常操作：打开图片 → 对着打字 → 打完还要检查有没有错别字。来回折腾两三分钟。

用 Quicker：鼠标中键一按 → 点「截图OCR」→ 框选文字区域 → 文字直接出现在剪贴板。全程不超过5秒。

再想象第二个画面。你有一个重要文件，但死活不记得放哪个文件夹了。

正常操作：打开电脑 → 资源管理器 → 一个文件夹一个文件夹翻 → 要么找到了（恭喜），要么放弃了（更可能）。

用 Quicker：鼠标中键一按 → 点「Everything搜索」→ 输入文件名前几个字 → 瞬间弹出结果。全程3秒。

**Quicker 就是这样一个东西**——它是一个 Windows 上的「指尖工具箱」。按一下鼠标中键，弹出一个面板，上面摆满你最常用的功能。点击就用，用完即走。

它不是普通的启动器，更像是一个**自动化平台**。上面有超过8000个现成的「动作」（相当于小插件），你只管装、只管用，不用写一行代码。

## 二、下载与安装：三分钟搞定

### 第一步：去哪里下载

Quicker 官网只有一个地址：

> **https://getquicker.net/**

打开之后点「下载」，你会看到三个版本：

| 版本 | 说明 |
|------|------|
| 64位版本 | 现在绝大多数电脑选这个 |
| 32位版本 | 老电脑（十年前那种）才选这个 |
| 快速通道/预览通道 | 普通用户不用管，选「普通通道」即可 |

不知道怎么选？默认下载64位就对了。

当前最新版本是 **v1.45.5**（2026年4月发布）。

### 第二步：安装

下载完是一个 `.exe` 安装文件，双击打开——

1. 选择安装路径（默认就行，不用改）
2. 一路点「下一步」
3. 安装完成

总共不超过30秒，没有捆绑软件，没有弹窗广告。

### 第三步：注册账号

Quicker **必须登录才能使用**。免费注册，一个邮箱就行。

安装完成后会自动弹出登录窗口。点「注册账号」→ 输入邮箱和密码 → 验证邮箱 → 完成。

为什么要登录？因为你的所有设置、动作、面板布局都会同步到云端。以后换了电脑，登录同一个账号，所有配置自动恢复，不需要重新设置。

### 第四步：开机自启

安装完 Quicker 会默认开机自启动。任务栏右下角会出现一个闪电图标⚡，说明它在后台运行了。

如果没看到，去 Quicker 设置里勾选「开机启动」即可。

## 三、认识 Quicker 的面板：你的指尖控制中心

### 怎么呼出面板

安装好之后，**按一下鼠标中键**（滚轮按下去），Quicker 面板就会弹出来，出现在鼠标光标的位置。

手指移动最短的距离，就能点到你想要的功能。

如果你的鼠标没有中键，或者不习惯，可以在设置里改成：
- 长按鼠标右键
- 鼠标侧键（如果鼠标有侧键的话，强烈推荐）
- 自定义键盘快捷键

### 面板长什么样

Quicker 的面板分为**上下两个部分**：

```
┌─────────────────────┐
│  ○  ○  ○  ○  ○  ○  ○  │  ← 上半部分：全局动作
│  ○  ○  ○  ○  ○  ○  ○  │     无论打开什么软件都会显示
│  ○  ○  ○  ○  ○  ○  ○  │
├─────────────────────┤
│  ○  ○  ○  ○  ○  ○  ○  │  ← 下半部分：上下文动作
│  ○  ○  ○  ○  ○  ○  ○  │     根据当前软件自动切换
│  ○  ○  ○  ○  ○  ○  ○  │
└─────────────────────┘
```

- **上半部分**叫「全局面板」：放的是不管在用什么软件都要用的功能，比如截图OCR、翻译、搜索、计算器……
- **下半部分**叫「上下文面板」：会根据你当前正在用的软件自动变化。比如你在Word里，下面就会出现Word专用的动作；切到浏览器，就变成浏览器专用的动作。

这个设计非常聪明——你不需要手动切换，Quicker 自动识别你在用什么软件，给你最适合的操作。

### 翻页功能

一个面板有28个格子（4行×7列），如果不够用，可以翻页：

- 点面板底部的小圆点翻页
- 或者用鼠标滚轮在面板上滚动翻页
- 免费版每应用可以翻1页，专业版不限

右下角还有一个小工具栏，放着设置、搜索、动作库入口等常用按钮。

### 自定义外观（专业版功能）

如果你不想千篇一律，Quicker 支持自定义面板外观：换背景图、换颜色、换字体大小、换深色模式……

面板外观也可以从动作库直接安装别人分享的，几秒钟换一套皮肤。

## 五、基础操作入门：从零搭建你的专属面板

### 5.1 添加常用软件捷径

这是最简单也是最实用的操作。

1. 在面板空白格子上**右键** → 选择「基础动作」→「打开软件」
2. 在弹出的窗口里找到你要添加的软件（比如微信、WPS、浏览器）
3. 给这个动作起个名字，选个图标
4. 点确定

之后每次想打开这个软件，弹出面板点一下就行。不用去桌面翻、不用去开始菜单找。

### 5.2 添加常用文件夹

每天要打开好几次的那个工作文件夹，也可以加到面板上。

1. 右键空白格子 → 「基础动作」→「打开文件夹」
2. 选择你的工作文件夹路径
3. 起名字、选图标
4. 确定

### 5.3 添加网址捷径

1. 右键空白格子 → 「基础动作」→「打开网址」
2. 输入网址（比如公司OA系统、内部知识库）
3. 搞定

### 5.4 从动作库安装动作（核心操作）

这才是 Quicker 真正的精髓——它有一个庞大的「动作库」，里面有上万个人家做好的功能，你直接拿来用。

操作步骤超级简单：

1. 打开 Quicker 面板 → 点右下角的「动作库」图标（或直接访问 getquicker.net/Share）
2. 在动作库里找到你想要的（后面第十章会推荐必装动作）
3. 点「复制」按钮
4. 回到 Quicker 面板 → 在空白格子上**粘贴**（Ctrl+V 或者右键粘贴）
5. 动作就装好了，直接能用

不需要下载文件、不需要解压、不需要配置。就是复制 + 粘贴，两步搞定。

### 5.5 调整动作位置

面板上的格子可以随意拖动：

- **拖到一个空格**：移动动作
- **拖到另一个有动作的格子**：交换位置
- **按住Ctrl拖**：复制动作到另一个格子
- **右键删除**：移除动作

排好位置，把最常用的放在最容易点到的区域。

## 六、十大必装热门动作推荐

以下是我精选的10个动作，每个都能实实在在帮你省时间。

### No.1 截图OCR —— 所有Quicker用户都会用的功能

**用途**：把图片、截图、扫描件里的文字「抠」出来，变成可复制粘贴的文字。

**使用场景**：
- 百度文库/某些网页禁止复制文字 → 截图 → OCR → 文字到手
- 视频教程里的网址/代码 → 截图 → OCR → 直接复制
- 扫描的PDF文档 → 截图 → OCR → 整理成文字
- 任何图片里的电话号码、身份证号 → 截图 → OCR

**操作**：弹出面板 → 点「截图OCR」→ 框选文字区域 → 文字自动识别并放入剪贴板。你甚至可以在识别后直接翻译。

**准确率极高**，中英文混排也能搞定。

### No.2 Everything搜索 —— 秒搜电脑里的任意文件

**用途**：比 Windows 自带搜索快100倍的本地文件搜索。

**为什么需要它**：Windows自带的搜索，搜一个文件名等半分钟还经常搜不到。Everything 用的是文件索引技术，输入文件名的前几个字，结果瞬间出现。

**Quicker版更强**：选中任何文字（比如一个文档里提到的文件名），点「Everything搜索」，直接搜。不需要打开Everything软件、不需要手动粘贴搜索词。

### No.3 EVER智识 —— 一个动作识别所有内容

**用途**：选中任何文字，自动判断它是什么、并执行对应操作。这是 Quicker 里的 **「万能动作」**。

**能识别的内容包括**：

| 选中内容 | 自动执行 |
|----------|----------|
| 网址 | 直接打开网页 |
| 数学算式 | 直接算出结果（如 `32-68*(50/6-3.28)+5`） |
| QQ号 | 直接打开对话框 |
| 快递单号 | 跳转查快递 |
| B站AV/BV号 | 打开视频 |
| 文件路径 | 打开对应路径 |
| 网盘链接 | 打开链接并自动填提取码 |
| 注册表路径 | 跳转注册表 |
| 其他文本 | 默认百度搜索 |

你不需要思考「这应该用什么工具」，选中 → 点 EVER智识 → 它自动帮你判断。

### No.4 剪贴板历史 —— 复制过的内容都能找回来

**用途**：记录你复制过的所有内容（文字、图片、文件），可以反复粘贴。

**典型场景**：写文档时需要从多个地方找资料，一会儿复制这个、一会儿复制那个，之前复制的又得重新去复制。有了剪贴板历史，所有复制过的内容都保存在这里，想用哪个点哪个。

**注意**：使用前要先在面板上点一下打开这个动作，它才开始记录。不想记录的时候关掉就行。

### No.5 至目录 —— 一步直达你想要的文件夹

**用途**：在任意软件的「保存」「另存为」「打开」对话框中，一键跳转到你收藏的常用文件夹。

**痛点场景**：很多软件（尤其是网盘客户端、某些办公软件）的「另存为」不记录上次位置，每次都要从「我的电脑」开始一层一层点进去。比如你想把文件存到 `D:\工作\2026年\6月\项目资料`，每次都要点至少4次。

用了「至目录」：收藏这个路径 → 在保存对话框里弹Quicker面板 → 点「至目录」→ 选收藏的路径 → 瞬间跳转。爽到飞起。

### No.6 快速翻译 —— 划词即译

选中英文 → 弹出面板 → 点翻译 → 中英对照结果。无需打开浏览器、无需粘贴到翻译网站。

支持多种翻译引擎，准确率比浏览器自带的好得多。

### No.7 批量重命名 —— 告别一个个改文件名

选中多个文件 → 弹出面板 → 点「批量重命名」→ 设置命名规则（添加前缀/后缀/序号/替换文字等）→ 一键改名。

整理照片、归类文档的时候，这个功能能省半小时。

### No.8 一键截图/长截图

- **截图**：比QQ截图更好用的截图工具，可以直接贴在屏幕上（悬浮）、可以标注、可以OCR
- **长截图**：滚动截取整个网页或长文档，自动拼接

### No.9 查快递

选中快递单号 → 弹面板 → 点查快递 → 自动识别快递公司并跳转查询结果。不需要先判断是哪家快递、再打开对应的网站。

### No.10 面板外观库

不想让面板永远是白底黑字？安装「外观库」动作，一键切换几十种面板皮肤。支持深色模式、渐变背景、透明效果……

## 七、进阶玩法：从「点一点」到「自动化」

当你熟练使用基础功能后，Quicker 还有这些更强的方式等待探索：

### 7.1 轮盘菜单（专业版）

按住鼠标一划，一个圆形菜单弹出，不同的方向对应不同的动作。不用精确点击、不用记位置，肌肉记忆形成后超级快。

比如：按住右键 → 往上一划 = 截图OCR，往左一划 = 翻译，往下一划 = 搜索……

### 7.2 鼠标手势（专业版）

在屏幕上画一个形状，触发对应动作。比如画个「√」是确认、画个「×」是关闭窗口。

### 7.3 扩展热键（专业版）

把普通键盘按键变成「修饰键」来组合快捷键。比如：
- 按着 `CapsLock` + `T` = 翻译
- 按着 `CapsLock` + `S` = 搜索

不用记 `Ctrl+Alt+Shift+某某某` 这种复杂组合，自己定义最顺手的。

### 7.4 文本指令

输入特定的几个字符，自动替换或触发操作。比如：
- 输入 `/addr` → 自动展开成你的完整地址
- 输入 `/now` → 自动插入当前日期时间
- 输入 `/mail` → 自动展开成你的邮箱

做客服、写邮件、填表单的时候特别有用。

### 7.5 组合动作：像搭积木一样做自动化

这是 Quicker 最强的能力——你可以把多个步骤串起来，变成一个一键执行的流程。

比如创建一个「下班关机流程」：
1. 保存所有打开的文档
2. 关闭所有软件
3. 清空剪贴板
4. 锁定屏幕

组合动作的编辑界面是可视化的，确实像搭积木——把这个模块拖过来、连上线、设置参数——不写代码也能搞定。

## 八、实战案例：四个我在工作中天天用的场景

### 场景1：写报告时，从PPT截图提取文字

Word里正在写报告，需要引用PPT里的某段话，但PPT是图片格式的。

**不用Quicker的做法**：截图 → 保存 → 打开看图软件 → 对着打字 → 核对 → 继续写报告

**用Quicker的做法**：鼠标中键 → 截图OCR → 框选 → 文字到手 → Ctrl+V 粘贴到文档

**用时对比**：2分钟 vs 5秒。

### 场景2：整理下载文件夹，50个文件需要分类重命名

从网上下载了一堆资料，文件名都是乱码或者无意义的数字。

**用Quicker的做法**：全选文件 → 弹面板 → 批量重命名 → 设置规则（比如「工作资料_001」「工作资料_002」……）→ 一键完成。

**用时对比**：手动一个个F2改名，至少10分钟 → 一键搞定30秒。

### 场景3：每天打开的8个工作文件夹

你的工作文件夹散落在各个盘里：`D:\工作\项目A`、`E:\备份\档案资料`、`F:\共享\团队文档`……

每次打开都要在资源管理器里翻半天。

**用Quicker的做法**：把这些文件夹全部加到面板上，一次设置终身受益。之后不管在哪个软件里、不管当前打开了什么窗口，弹出面板一点就到。

### 场景4：浏览器看到一段有用的话，想记下来

看到一段好内容，想保存到本地。

**用Quicker的做法**：选中文字 → 弹面板 → 点「添加到记事」或「保存为txt」→ 几秒钟搞定，不用切换软件。

## 九、常见问题 FAQ

### Q1：免费版够用吗？

对于80%的普通用户来说，**完全够用**。

免费版限制的核心是：只能为10个软件设置专属动作、每软件只有1页动作、OCR识别有次数限制、轮盘菜单只有两圈。

如果你不是重度用户，这些限制几乎碰不到。先用免费版，真的觉得不够了再升级。

### Q2：Quicker 安全吗？我的数据会被泄露吗？

Quicker 不收集你的文件内容。需要联网是因为：
- 同步你的面板配置到云端（你自己的数据，加密存储）
- 动作库需要联网浏览和下载
- OCR功能需要在线识别引擎

敏感操作（比如密码输入）不会通过 Quicker 传输。动作库里的动作代码是公开透明的，不会偷偷上传你的隐私数据。

### Q3：怎么备份我的设置？

Quicker 的设置会自动同步到云端，登录同一个账号即可恢复。

如果需要手动备份：设置 → 备份与恢复 → 导出全部数据。导出的文件可以存到U盘或网盘。

专业版还有自动历史版本功能，可以恢复到之前的某个状态。

### Q4：换了新电脑怎么办？

在新电脑上安装 Quicker → 登录同一个账号 → 所有设置、动作、面板布局自动恢复。

像什么都没发生一样。

### Q5：Quicker 和 WorkBuddy 有什么区别？是不是功能重叠？

两个完全不同定位的工具，互补关系：

| | Quicker | WorkBuddy |
|------|---------|-----------|
| 定位 | 操作加速器 | AI办公助手 |
| 主要方式 | 点一下，快速完成预定义操作 | 说一句话，AI理解意图并执行 |
| 更适合 | 重复性高频操作（截图OCR、开软件、搜文件） | 需要思考判断的复杂任务（写文章、分析数据、做PPT） |
| 学习成本 | 极低（安装动作就能用） | 需学会如何表达需求 |

**一个是「快」，一个是「智」。两个一起用，效率翻倍。**

### Q6：有些高级功能我不会用怎么办？

Quicker 官网有详细的中文文档（getquicker.net/KC），还有视频教程。

动作库里每个动作都有使用说明，作者通常会写清楚怎么操作。

如果还不会，在 Quicker 社区论坛提问，回应很快。

### Q7：Mac 能用吗？

Quicker 目前只支持 **Windows** 平台。Mac 用户可以考虑替代方案（如 Alfred、Raycast），但 Quicker 的动作库生态目前是独一档的。
