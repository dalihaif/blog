## 💡 使用建议

### 写作流程

1. **规划内容**
   - 确定文章主题
   - 准备相关图片素材
   - 构思文章结构

2. **创建文章**
   ```markdown
   ---
   layout: post
   title: "文章标题"
   subtitle: "副标题"
   date: 2024-01-15
   author: "老墨"
   header-img: "img/your-image.jpg"
   tags:
     - 标签1
     - 标签2
   ---
   
   文章内容...
   ```

3. **本地预览**
   - 运行 `启动博客.bat`
   - 检查显示效果
   - 调整格式和内容

4. **提交发布**
   ```bash
   git add .
   git commit -m "新增文章：XXX"
   git push
   ```

### 维护建议

- **定期更新**：每周至少发布一篇文章
- **互动回复**：及时回复读者评论
- **持续优化**：根据反馈改进网站体验
- **备份数据**：定期备份文章和图片
